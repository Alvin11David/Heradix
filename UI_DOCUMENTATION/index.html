📍 User Journeys

🔄 User Signup & Login Journey

Complete flow from landing to authenticated user

/ → /auth/login → /auth/register → /auth/forgot-password
Components & Features:
  • Marketplace (Asset List): Browse assets, Search, Filter by format/category
  • Login: Email/Password login, Remember me, Forgot password link
  • Register: Email signup, Password validation, Terms acceptance
  • Forgot Password: Email recovery, Password reset

🔄 Marketplace Asset Discovery

Browse, search, and view asset details

/marketplace → /marketplace/asset/:slug
Components & Features:
  • Asset List: Grid view, Search bar, Filters, Pagination
  • Asset Detail: Preview, Download formats, Similar assets, Share options

🔄 Collections Management

Organize and manage user collections (Premium)

/collections → /collections/:id
Components & Features:
  • Collections List: View all collections, 2x2 mosaic preview, Search collections
  • Collection Detail: View all items in collection, Click item to view asset detail
