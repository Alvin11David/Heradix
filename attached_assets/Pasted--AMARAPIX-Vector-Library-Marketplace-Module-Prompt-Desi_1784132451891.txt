# AMARAPIX Vector Library & Marketplace Module Prompt

Design and implement a **world-class Vector Library and Marketplace** for AMARAPIX that rivals or surpasses Canva, Freepik, Vecteezy, Adobe Stock, SVG Repo, Icons8, and Storyset. The experience should be modern, premium, fast, AI-powered, scalable, and highly intuitive.

---

# General Requirements

Create a dedicated **Vectors** section where users can discover, search, edit, organize, download, upload, share, and use millions of vector assets directly inside the AMARAPIX editor.

The system should support:

* SVG
* EPS
* AI
* PDF
* CDR
* DXF
* PNG Preview

Every vector should be editable directly inside the editor without leaving the application.

* Dark and light themes
* Keyboard shortcuts
* Drag-and-drop interactions
* Context menus
* Breadcrumb navigation
* Powerful filtering and sorting
* Accessible design (WCAG compliant)
* Fast page transitions
* Rich previews and micro-interactions

The final experience should combine the best ideas from Canva, Freepik, Vecteezy, SVG Repo, Adobe Stock, Storyset, Icons8, and modern AI-powered creative tools, while remaining seamless, scalable, and deeply integrated with every other module in the AMARAPIX ecosystem.
