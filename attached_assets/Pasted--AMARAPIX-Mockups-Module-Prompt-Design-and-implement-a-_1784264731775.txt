# AMARAPIX Mockups Module Prompt

Design and implement a **world-class Mockups module** for AMARAPIX that rivals or surpasses Canva, Placeit, Smartmockups, Freepik Mockups, Envato Elements, Adobe Express, and Artboard Studio.

The Mockups section should allow users to instantly preview their designs on realistic products, devices, apparel, packaging, signage, print materials, and digital screens using smart object technology and AI-powered placement.

---

# General Requirements

Create a dedicated **Mockups** section where users can:

* Browse thousands of professional mockups
* Search mockups instantly
* Preview mockups
* Replace designs using Smart Objects
* Edit backgrounds
* Edit shadows
* Customize scenes
* Generate AI mockups
* Download high-quality renders
* Save mockups to collections
* Use mockups directly inside the AMARAPIX editor

The experience should be modern, premium, and extremely easy to use.
