# Mock Asset Images

Place your mock thumbnail images here.

## Usage in templates
Reference images as: `/assets/images/your-file.jpg`

## Recommended structure
```
images/
  thumbnails/     ← asset card cover images (recommended 400×300px, JPG/WebP)
  banners/        ← hero / promotional banners
  avatars/        ← user avatar placeholders
```

## Supported formats
JPG, PNG, WebP, SVG — all are served statically by Angular's dev server.
